# 隐私政策 / Privacy Policy — vcf-rosetta 浏览器扩展

_最后更新:2026-06-24_

## 简体中文
vcf-rosetta 是一个纯前端的界面本地化(翻译)扩展。

- **不收集任何个人信息。** 扩展不读取、不存储、不传输用户身份、凭据、浏览历史或页面业务数据。
- **不向任何服务器发送页面内容。** 翻译在浏览器本地完成,词典文件随扩展打包或按需下载到本地后使用。
- **本地设置。** 仅在浏览器本地 / 账号同步存储区(`chrome.storage`)保存用户偏好:启用开关、界面语言、生效主机白名单、是否开启"未翻译词条采集"。这些数据不发送给作者或第三方。
- **未翻译词条采集(默认关闭,需用户手动开启)。** 开启后,扩展会在本地收集当前页面上"未命中词典的英文界面文本"。这些文本仅保存在内存中,由用户主动点击"导出"生成 JSON 文件下载到本地。**扩展本身不会自动上传任何内容**;是否将导出文件提交给项目,完全由用户自行决定。
- **无远程分析、无广告、无跟踪。**

## English
vcf-rosetta is a client-side UI localization (translation) extension.

- **No personal data is collected.** It does not read, store, or transmit identity, credentials, browsing history, or business data from pages.
- **No page content is sent to any server.** Translation happens locally using dictionary files bundled with the extension or downloaded on demand to the local device.
- **Local settings only** are kept in `chrome.storage` (enable toggle, UI language, host allow-list, optional "collect untranslated strings" flag). They are never sent to the author or third parties.
- **Untranslated-string collection (off by default, user-enabled).** When enabled, untranslated English UI strings on the current page are gathered in memory and only written to a local JSON file when the user clicks "export". The extension never auto-uploads anything; submitting an exported file to the project is entirely the user's choice.
- **No remote analytics, ads, or tracking.**
