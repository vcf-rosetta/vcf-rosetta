VMware Skill Family for LLMs

The same team behind this translator builds an open-source VMware skill family — drive vCenter, ESXi, NSX, vSAN and Aria in natural language with any LLM, cloud-hosted or fully local. vmware-aiops is the entry point; add the others as you need them.

Skill Scope Install

Quick install

Requires uv (or pip). vmware-aiops is the recommended entry point; each skill is an independent MCP server.

uv tool install vmware-aiops
vmware-aiops mcp
# Claude Code plugin
/plugin marketplace add zw008/VMware-AIops
/plugin install vmware-aiops
